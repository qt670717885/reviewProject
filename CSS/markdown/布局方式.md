- 两栏布局,左侧栏宽度固定，右侧自适应

  - 方法一: float+margin

    ```html
     <div class="left"></div>
     <div class="right"></div>
    ```

    ```css
    .left {
       float: left;
       width: 200px;
       height: 100vh;
       background-color:pink
    }
    
    .right {
       height: 100vh;
       background-color:tomato;
       margin-left:200px;
    }
    ```

    

  - 方法二： position:absolute+margin

  ```html
   <div class="left"></div>
   <div class="right"></div>
  ```

  ```
  .left {
     position:absolute;
     width: 200px;
     height: 100vh;
     background-color:pink
  }
  
  .right {
     height: 100vh;
     background-color:tomato;
     margin-left:200px;
  }
  ```

  - 方法三: flex

  ```html
   <div class="warp">
      <div class="left"></div>
      <div class="right"></div>
  </div>
  ```

  ```
  .warp{
      display: flex;
  }
  .left{
      width: 200px;
      background: tomato;
      height: 100vh;
  }
  .right{
      background: pink;
      flex: 1;
  }
  ```



- 三栏布局: 两端固定，中间自适应

  - 方法一: 使用float+margin的方式

  ```html
  <div class="wrap">
          <div class="left"></div>
          <div class="middle"></div>
          <div class="right"></div>
  </div>
  ```

  ```css
  		.left,.right{
              width: 200px;
              height: 100vh;
              background: tomato;
          }
          .middle{
              margin: 0 200px;
              background: pink;
              height: 100vh;
          }
          .left{
              float: left;
          }
          .right{
              float: right;
          }
  ```

  - 方法二 : flex

  ```html
  <div class="wrap">
          <div class="left"></div>
          <div class="middle"></div>
          <div class="right"></div>
  </div>
  ```

  ```css
  		.left,.right{
              width: 200px;
              height: 100vh;
              background: tomato;
          }
          .wrap{
              display: flex;
          }
          .middle{
              flex: 1;
              background: pink;
          }
  ```



- 三栏布局:上下固定，中间自适应

  - 方法一: 利用定位，position：fixed+position:absolute

  ```html
  <div class="top"></div>
  <div class="content"></div>
  <div class="bottom"></div>
  ```

  ```css
  		*{
              margin: 0;
              padding: 0;
          }
          .top,.bottom{
              position: fixed;
              width: 100%;
              height: 100px;
              background: tomato;
          }
          .top{
              top: 0;
          }
          .bottom{
              bottom: 0;
          }
          .content{
              background: pink;
              position: absolute;
              top: 100px;
              left: 0;
              right: 0;
              bottom: 100px;
              overflow: auto;
          }
  ```

  - 方法二:flex

  ```
  	<div class="warp">
          <div class="top"></div>
          <div class="content"></div>
          <div class="bottom"></div>
      </div>
  ```

  ```
  		html,body{
              width: 100%;
              height: 100%;
          }
          .warp{
              height: 100%;
              display: flex;
              flex-flow: column;
          }
          .top,.bottom{
              width: 100%;
              height: 100px;
              background: pink;
          }
          .content{
              flex: 1 1;
              background: tomato;
          }
  ```



双飞翼布局与圣杯布局解决的问题都是一样的，都是两边顶宽，中间自适应的三栏布局(中间栏放在文档流前面有限渲染)

两者的区别：

​	圣杯布局和双飞翼布局解决问题的方案在前一半是相同的：
​	也就是三栏**全部浮动**，但左右两栏加上**负margin**让其跟中间栏div并排。

​	不同在于解决 **“中间栏div内容不被遮挡”** 问题的思路不一样。

- 圣杯布局:使用的是padding压缩空间后使用定位将位置移动到恰当位置

- 双飞翼布局:使用的在main外层多家一个div改用margin

```html
	//圣杯布局
	<header>header</header>
        <div class="container">
            <div class="main">main</div>
            <div class="left">left</div>
            <div class="right">right</div>
        </div>
    <footer>footer</footer>
```

```css
 		* {
            margin: 0;
            padding: 0;
        }

        header,
        footer {
            width: 100%;
            background: green;
        }

        .container {
            background: gray;
            overflow: hidden;
            padding: 0 200px;
            min-width: 200px;
        }

        .main,
        .left,
        .right {
            float: left;
            min-height: 100px;
        }

        .main {
            background: yellow;
            width: 100%;
        }

        .left {
            background: red;
            margin-left: -100%;
            width: 200px;
            position: relative;
            left: -200px;
        }

        .right {
            background: red;
            margin-left: -200px;
            width: 200px;
            position: relative;
            right: -200px;
        }
```

```html
//双飞翼布局
```
